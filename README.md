# Readme



■ゲームの概要



【ゲームタイトル】Apple Catch

【ジャンル】シューティング

【特徴】連射による爽快感、時間制限による緊張感

【難易度】簡単

【プレイ時間】１～２分

【制作ソフト】DxLib

【使用言語】C++

【ゲーム制作者】嶋田瑛心



---

■ゲームの起動方法



zipファイルを展開し、「Apple Catch.exe」ファイルをダブルクリックしてください。



---

■操作方法



▼キーボード

\[矢印キー]：移動

\[Enter]：決定

\[Space]：発射

\[Esc]：ポーズ（メニュー）



▼マウス

\[左クリック]：選択



▼ゲームパッド（Nintendo Switchのボタン配置基準）



\[十字キー][左スティック]：移動

\[Aボタン]：決定・発射

\[Bボタン]：戻る

\[Xボタン]；ポーズ（メニュー）



---

■注意点



音声が全体的に少々大きめになっています。

そのため、PC本体の音量を10前後から始めて、

調節してください。ゲーム内に音量を変更する

仕組みはありません。
